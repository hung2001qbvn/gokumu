Gomoku game with Adversarial searching
=======   

## Usage
Clone our source code
```sh
$ git clone https://github.com/hung2001qbvn/gokumu.git
$ cd gomoku
```

Install Dependencies

```sh
$ pip install -r requirements.txt
```

Install Pygame
```sh
$ pip install pygame
```

Run program

```sh
$ python main.py
```
